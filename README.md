# ParticleNeoPixelHelloWorld
Basic program for chaining and testing Adafruit Flora Neopixel v2 on a Particle Photon.

![Working Picture](neopixel-working.JPG "Working Picture")

# Connections

NeoPixel | Photon

(-) --> GND

(+) --> VIN

(->) --> D2
